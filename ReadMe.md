## 关于本项目

一个读取系统信息以及硬件使用记录的工具。

## 功能

本程序支持的功能有：

- [x] 读取电脑当前系统的系统信息
- [x] 安装的软件列表
- [x] 使用过的CPU记录
- [x] 使用过的GPU记录
- [x] 使用过的存储设备记录
- [x] 使用过的主板/BIOS记录
- [x] 使用过的网卡记录
- [x] 使用过的音频设备记录
